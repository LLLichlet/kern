1. 更好的边界体验。
```rust
println("Smashing stack with: {} {} {}", .{trash1.&, trash2.&, trash3.&}) -> 
println("Smashing stack with: {} {} {}", .{trash1, trash2, trash3})
```
边界退化规则依据`trash1`等value是否是`let mut`来定，并只允许非指针应用这种规则。
同时把约束应用到闭包上。
```rust
array.sort(.[](a: i32, b: i32) i32 { a >= b });
```
这种情况是“无类型”，所以直接可以到闭包。
```rust
let sorter = .[](a: i32, b: i32) i32 { a >= b };
array.sort(sorter);
```
这种情况要看。比如这里是`let sorter`，所以兼容的是`*Fn(i32, i32) i32`，而不兼容`*mut Fn(i32, i32) i32`

同理如`TraitObj`
