# Kern Brand Assets

This directory contains the canonical Kern visual assets.

- `kern-logo.svg`: full logo for README and documentation surfaces.
- `kern-mark.svg`: square mark for editor language and file icons.
- `kern-mark.png`: PNG rendering of the mark for packaged extension metadata.

Generated or packaged copies may live under tool-specific directories, such as
`editors/vscode/icons/`, but those copies should be refreshed from this
directory rather than edited directly.
